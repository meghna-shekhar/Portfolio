Project

CareerQuest.in – Course Recommendation Website

Web Development · Recommendations · Web Scraping

Overview

CareerQuest.in is a course-recommendation website that helps students discover online courses aligned with their career goals. It scrapes course data from multiple MOOC providers, stores it in a custom SQL database, and lets users compare options side by side.

Why I built it

When I was exploring online courses, I had to jump between platforms, search manually, and compare prices and ratings by hand. CareerQuest.in was my way of turning that pain point into a project: build a small system that centralizes course information and lets users filter it by subject and career path.

How it works

  1. Scraping & ingestion: scripts collect course titles, providers, categories, prices, and ratings from different MOOC platforms.
  2. Database: data is stored in a SQL schema keyed by subject, career path, and provider, enabling flexible queries.
  3. Web UI: users choose a domain or career path (e.g., data science, web development) and see curated course lists.
  4. Comparison: the interface highlights ratings, price, and key details so users can compare courses across platforms at a glance.

Front-end

The site is built with HTML, CSS, JavaScript, Bootstrap, and jQuery. The focus was on a clean, simple layout that works well on both desktop and smaller screens.

Back-end & data

On the back-end, I designed a straightforward SQL schema for:

  • Courses (title, description, provider, price, rating, URL).
  • Categories and career paths.
  • Mappings between courses and career domains.

This made it easy to support queries like “show data-science courses under \$X with rating ≥ Y”.

Takeaways

  • Hands-on experience with web scraping and normalizing messy data.
  • Designing simple but useful recommendation logic backed by SQL.
  • Full-stack thinking: from scripts and database design to UI and UX.

Tech stack

Front-end: HTML, CSS, Bootstrap, JavaScript, jQuery
Back-end & data: Python (for scraping), SQL database
Topics: Web scraping, ranking & recommendation, full-stack web dev
