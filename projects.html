Featured & Recent

Personalized Story Book Generation with Fine-tuned LLM

Multimodal AI · Education · LLMs

Cost-efficient multimodal system that generates personalized storybooks for K–2 learners using a fine-tuned open-source LLM and UFLI-aligned prompts. A custom vision pipeline ensures consistent, age-appropriate illustrations, with all assets stored and served via Supabase for classroom deployment.

UrbanTwin AI – Generative City Simulation

Graph Neural Networks · GeoSpatial · Hackathon

AI-powered city twin built for UBHackopoly 2025 that lets users explore how changes in building density, green cover, and traffic affect urban heat islands, congestion, and air quality. Combines STGNNs, LightGBM, and Vertex AI (Gemini 2.5 Pro) with OpenStreetMap + MODIS data and an interactive dashboard.

Buffalo Accident Risk Prediction & Resource Allocation (RL)

Reinforcement Learning · City Analytics

PPO-based reinforcement learning system that predicts high-risk accident zones from Buffalo traffic data and optimizes deployment of emergency vehicles. Includes geospatial heatmaps (ZIP code & 500m grids) and comparisons against a random baseline agent.

Skin Disease Detection & Classification

Computer Vision · Medical Imaging

Deep learning pipeline using VGG19, ResNet50, EfficientNet-B0, and DenseNet121 to classify five common skin diseases. Built an ensemble by averaging softmax outputs to improve stability, achieving up to 74% test accuracy and LAB-based overlays for lesion localization and explainability.

Signature Projects

Speech to Indian Sign Language using 3D Pose Estimation

Computer Vision · NLP · 3D Pose

End-to-end translation medium from text/speech to Indian Sign Language using 2D pose estimation from ISL videos, depth regression to 3D pose, and grammar-aware text processing with NLTK. Outputs motion-capture (.bvh) files rendered in Blender for ISL animation.

NamaSign: Indian Sign Language Alphabet Detection

Deep Learning · Image Classification

Image classifier for ISL alphabet hand signs using VGG16 and ResNet18 on a real-world, cluttered dataset with varied skin tones, backgrounds, and lighting. Achieved 93.9% and 91.8% accuracy respectively.

CareerQuest.in – MOOC Recommendation Platform

Web Development · Recommender Systems

Web app that scrapes online course platforms and recommends programs based on user-chosen career paths. Supports keyword-based matching, custom SQL backend, and side-by-side comparison of course prices, structures, and reviews.

Covid-19 Data Visualization for AIIMS Delhi

Data Visualization · MongoDB · Grafana

Built dashboards in Grafana over MongoDB to differentiate pneumonia vs. COVID-affected lung X-rays and provide actionable logistical insights to AIIMS Delhi through interactive charts and alerts.
