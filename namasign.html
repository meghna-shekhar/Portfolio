Project

NamaSign: Indian Sign Language Alphabet Detection

Deep Learning · Image Classification · Accessibility

Overview

NamaSign is an image classification model for Indian Sign Language (ISL) alphabets. The goal was to recognize static hand poses in realistic conditions – different skin tones, backgrounds, camera distances, and lighting – rather than a clean lab dataset.

Problem & motivation

Many ISL alphabet datasets are captured in controlled environments. Real users sign in front of messy backgrounds with varying illumination, which makes recognition harder. This project explores how far off-the-shelf convolutional networks like VGG16 and ResNet18 can go on a noisy, practical ISL alphabet dataset.

Dataset

I built a cluttered ISL alphabet dataset by collecting images across:

  • Multiple signers with different skin tones and hand shapes.
  • Varied backgrounds (indoor, outdoor, textured walls, furniture, etc.).
  • Non-uniform lighting and camera angles.

This forces the model to learn shape and pose rather than memorizing a single clean background.

Approach

The pipeline is built around transfer learning:

  • Preprocess images (resize, normalization, augmentation with flips, jitter, and crops).
  • Fine-tune VGG16 and ResNet18 (FastAI) on the alphabet classes.
  • Use data augmentation aggressively to simulate new backgrounds and lighting changes.
  • Evaluate per-class performance to spot confusing signs.

Results

  • VGG16 achieved ≈93.9% test accuracy.
  • ResNet18 achieved ≈91.8% test accuracy.
  • Most errors occurred between visually similar hand shapes under extreme lighting.

Overall, the project shows that transfer-learning baselines, combined with strong augmentation, can handle noisy ISL alphabet data surprisingly well.

What I learned

I gained experience in building image datasets from scratch, designing augmentations that reflect real-world sign usage, and comparing different CNN architectures beyond just accuracy (e.g., robustness to distribution shift).

Tech stack

Languages: Python
Libraries: PyTorch / FastAI, TensorFlow / Keras, OpenCV, NumPy
Topics: Transfer learning, image augmentation, multi-class classification
